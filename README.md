# Password-Game
A password hacking game where you guess a (completely made up) user's password

Work in progress 🚧
This project is not finished yet.

How to play: 
You will be offered 5 user profiles and their usernames, the users password will be something that can be deduced from their profile, for example, their pet's name, date of birth, or something about their family.

Also remember to use capitals and correct spelling (for example, 'Thomas' or 'Sienna' not 'thomas' or 'Siena').

There are different levels of how secure a password is, for example:

Level 1: Charlie
Level 2: Charl13!
Level 3: Charl132017!

Enjoy the game, and please do not attempt this in real life, as it is illegal and unethical.

Seriously! Keep it in the game!
